## [C. Anti-Division](https://atcoder.jp/contests/abc131/tasks/abc131_c)

数据范围：

* $1\le A\le B \le 10^{18}$
* $1 \le C,D\le 10^9$

考察前缀和、容斥原理。一般区间问题都往前缀和里靠，求 $[a,b]$ 范围内不是c也不是d的倍数可以转化为 $[1,b]$ 内符合条件的值减去 $[1,a-1]$ 内符合条件的值。

那这样就变得很简单了，因为要求区间的倍数（约数）个数直接整除就行了。比如 $[1, 7]$ 2的倍数的话就是 $\lfloor \frac{7}{2} \rfloor$ 。

因为 $c$ 和 $d$ 可能有最小公倍数会多减掉一些，所以最后还要在加上最小公倍数的个数。

容斥原理的话其实就是正难则反，但求还是使用前缀和思想求。
