## 数据范围

* $1 \le N\le 10^5$
* $0\le A_i,B_j,C_k \le10^9$

## 思路

考察枚举优化，利用同余加法减小枚举空间。



$$
(A_i+B_j+C_k)\pmod{46}=((A_i\bmod46)+(B_j\bmod46)+(C_k\bmod46))\pmod{46}
$$



所以很多数可以不用遍历，它们模 $46$ 都余数都相同，只需要统计所有元素模 $46$ 的余数的频率即可。 再乘法原理相加即可得到答案。

```python
ans = 0
for i in range(46):
    for j in range(46):
        for k in range(46):
            if (i + j + k) % 46 == 0:
                ans += (cntA[i] * cntB[j] * cntC[k])
```
