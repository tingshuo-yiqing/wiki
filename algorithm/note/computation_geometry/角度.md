### 1. 核心概念：角度 vs 弧度

在数学和编程中，绝大多数三角函数库（包括 Python 的 `math` 库）默认接受的输入和返回的输出都是**弧度**，而不是我们日常习惯的角度。

- **角度 (Degrees):** 一个圆周分为 $360^\circ$。
- **弧度 (Radians):** 一个圆周分为 $2\pi$ (约 6.28)。

**换算公式：**


$$
\text{rad} = \text{deg} \times \frac{\pi}{180}
$$

$$
\text{deg} = \text{rad} \times \frac{180}{\pi}
$$

#### 三角函数 (输入均为弧度)

> **SOH – CAH – TOA**

记忆方式：

- SOH：**S**in = **O**pp / **H**yp
- CAH：**C**os = **A**dj / **H**yp
- TOA：**T**an = **O**pp / **A**dj

#### 反三角函数 (返回均为弧度)

- `math.asin(x)`: 反正弦，返回范围 $[-\pi/2, \pi/2]$
- `math.acos(x)`: 反余弦，返回范围 $[0, \pi]$
- `math.atan(x)`: 反正切，返回范围 $[-\pi/2, \pi/2]$

三角函数和反三角函数是互逆运算，它们前者**已知角度求比值**，后者**已知比值求角度**。

### 2. `math.atan2(y, x)`

普通的 `math.atan(y/x)` 有一个致命缺陷：它无法区分象限。例如，点 $(1, 1)$ 和点 $(-1, -1)$ 的斜率都是 $1$，`atan` 会对两者都返回 $45^\circ$ ($\pi/4$)，但实际上后者应该是 $225^\circ$ ($-135^\circ$)。

**`math.atan2(y, x)`** 解决了这个问题：

1. **参数：** 接受 $(y, x)$ 两个坐标（**注意 y 在前**）。
2. **范围：** 返回值范围是 $(-\pi, \pi]$。
3. **象限感知：** 它能正确处理所有四个象限以及 $x=0$ 的情况。

**示例：**

```python
y, x = -1, -1
# 错误的做法
# angle = math.atan(y/x) # 得到 0.785 (45度), 丢失了象限信息

# 正确的做法
angle = math.atan2(y, x) 
print(angle) # -2.356... (即 -3pi/4 或 -135度)
print(math.degrees(angle)) # -135.0
```
