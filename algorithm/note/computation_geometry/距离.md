## 欧几里得距离

Python 3.8+ 以上推荐使用 `math.dist()` 内置函数实现。

接收两个点，这两个点可以是列表元组等可迭代对象，并返回它们之间的欧几里得距离。使用简单方便，返回 `flaot` ，可以处理多维距离。

3.8 以前使用 `math.hypot()` ，它最初用来计算直角三角形的斜边长。通过对两点的左边作差可以构造一个直角三角形。

```python
import math

p1 = (1, 2)
p2 = (4, 6)

# 计算 (x2-x1, y2-y1) 的模
distance = math.hypot(p2[0]-p1[0], p2[1]-p1[1])
print(distance)  # 输出: 5.0
```
