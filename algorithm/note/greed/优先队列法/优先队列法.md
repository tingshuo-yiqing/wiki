[和并果子](https://www.luogu.com.cn/problem/P1090)

## 模板

注意`heapq`库的使用

```python
import heapq

n = int(input())

nums = list(map(int, input().split()))

heapq.heapify(nums)

ans = 0

while len(nums) > 1:
    a = heapq.heappop(nums)
    b = heapq.heappop(nums)
    heapq.heappush(nums, a + b)
    ans += (a + b)

print(ans)
```
