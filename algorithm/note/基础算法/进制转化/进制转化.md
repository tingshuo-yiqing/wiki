### 模板

**除基取余法**将 $x$ 进制转换为 $base$ 进制：

```python
def trans(x, n, base):
    num = int(n, x)  
    if num == 0: return '0'

    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    res = []

    while num:
        num, r = divmod(num, base)
        res.append(digits[r])

    return ''.join(res[::-1])
```

关键步骤：

* `digits` 数组，用于字符映射。
* `int(string, n)` 用于将 `n` 进制的字符串 `string` 转化为10进制。
* `divmod` 返回 (num // base, num % base)

**递归法**将 $10$ 进制转换为 $base$ 进制:

```python
def fast_convert(n, b):
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    return (fast_convert(n // b, b) + digits[n % b]) if n else ""
```
