一般在操作整数的二进制位时，从 $0$ 开始计数且从右往左看。

1. `&` 两个都是 1 才得 1 ，常用于**取位**
2. `|` 有 1 为 1 ，常用于**置位**（设为1）
3. `^` 不同得 1 ，常用于**翻转**和**不进位加法**
4. `~` 取反
5. `<<` 可以理解为乘以 $2^i$
6. `>>` 可以理解为整除 $2_i$

## 基础操作

判断 $n$ 的第 $i$ 位是 0 还是 1

```python
(n >> i) & 1 
n & (1 << i) # 两种方法都可以
```

修改第 $i$ 位

* `n | (1 << i)` 将第 $i$ 位设置为 1
* `n & ~(1 << i)` 将第 $i$ 位设置为 0
* `n ^ (1 << i)`  翻转第 $i$ 位

### LSB

最左边的 1 决定了数值的数量级（即 `bit_length`）

获取最左边的 1 可以直接使用 Pyhton 内置的 `bit_length()` 函数。也可以自己手写 while 循环实现。

```python
def LSB(n):
    cnt = 0
    while 1 << cnt <= n:
        cnt += 1
    return cnt
```



### RSB

最右边的 1 含有很多性质。

可以通过 `x & -x` 来获取最右边的 1
