### **B-Card Game**

因为比不过不会被移除，所以我方最大的一定可以等到对方较小的而实现加分，于是最大得分就是大于 $\min(B)$ 的牌数 $K$ 。

又比 $\min(B)$ 还小的永远无法上场所以可以随便排，根据乘法原理可知：


$$
ans = K!\times(n-K)!\pmod{998244353}
$$


建议：在处理多次的阶乘查询时可以预处理阶乘数组

```python
# 预处理阶乘
fact = [1] * (MAX_N + 1)
for i in range(2, MAX_N + 1):
    fact[i] = (fact[i-1] * i) % MOD
```

### **C-Array Covering**

选择一个区间 $(l,r)$ 将区间除端点外的点都变成较大的那个端点。不难看出选择除端点的剩余元素的最大值即可。


$$
ans = \max(a[1:-1]) + a[0]+a[-1]
$$
