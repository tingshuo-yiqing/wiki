## 赛后复盘

过了ABC。

AB有思路，A很快就构造出来了，B的条件有点难满足写了挺久，C想了很久但理解了后也很简单。

### **A. Divisible Permutation**

构造一个 $|a_i-a_{i+1}|$ 可以被 $i$ 整除的序列，不难发现我们可以构造一个形如 $···,n-2,3,n-1,2,n,1$ 的序列，恰好可以满足条件。

具体的为构造 $1,n,2,n-1,3,n-2,···$ 再反转。

### **B. Seats**

不难看出三个空座位坐一个人可以最小化人数，整体思路是连续的空座位对 $3$ 进行上整除。

* 当全为 $0$ 时，直接对 $3$ 上整除
* 含有 $1$ 时，先让 $ans = s.count('1')$ 再看连续的 $0$ 两端的 $1$ 的数量，两端有多少个 $1$ 就减去多少。

当然，也可以使用哨兵将逻辑整合，在前期对字符串进行 $'1' + s + '1'$ 来避免分类讨论，只需要在计算答案时判断一下这个 $1$ 是不是两端的。

### **C. Restricted Sorting**

先特判已经单调的情况，再使用一个排序后的数组进行一一对比看看哪些元素错位了。如果要让该错位的元素交换的话那它的 $k$ 值区间一定在 $|\max(a) - a_i|$ 和 $|a_i-\min(a)|$ 之间。

所以只需要收集所有错位元素的 $k$ 值区间在取 $\max$ 即可。
